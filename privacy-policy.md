# Privacy Policy

## Introduction
This application uses the WhatsApp Cloud API to receive and send messages automatically.

## Data Collected
We collect only the data necessary to operate the service:
- WhatsApp phone number
- Message content sent by the user

## Data Usage
The collected data is used solely to:
- Respond to user messages
- Automate workflows via n8n

## Data Storage
Messages are processed in real time and are not stored permanently.

## Data Sharing
We do not share user data with third parties.

## User Rights
Users may stop interacting with the service at any time by discontinuing messages.
